Technologies Used: ☕️ 🐍 ⚛️
- react native in Typescript 
- redux
- redux saga
- some other ui component library e.g navigation,treeView

Setup / Installation: 💻
after checkout from github

assuming yarn is installed 
```
yarn install
yarn ios #for ios 
yarn android #for android
```
